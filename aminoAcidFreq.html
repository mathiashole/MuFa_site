4.1 üß¨ An√°lisis de la composici√≥n de amino√°cidos

Los amino√°cidos son los componentes fundamentales de las prote√≠nas, y su composici√≥n puede ofrecer informaci√≥n valiosa sobre las propiedades funcionales, estructurales y evolutivas de las secuencias codificantes. üîç En este an√°lisis se comparan dos grupos de secuencias:

üß™ Secuencias DGF-1: prote√≠nas pertenecientes a la familia multig√©nica DGF-1 de Trypanosoma cruzi. Esta familia se ha asociado con procesos de interacci√≥n par√°sito-hospedador y presenta una alta variabilidad, por lo que resulta de especial inter√©s biol√≥gico.

üß¨ Prote√≠nas codificantes del genoma: conjunto de prote√≠nas anotadas como codificantes en los genomas de referencia, usadas como contexto comparativo.

A partir de ambas fuentes, se calcularon las frecuencias relativas de cada amino√°cido en cada prote√≠na. Luego, estas frecuencias fueron transformadas a porcentajes para facilitar la comparaci√≥n, y se calcularon estad√≠sticas resumen (media y desviaci√≥n est√°ndar) por grupo.

El objetivo es evaluar si las prote√≠nas DGF-1 presentan un perfil distintivo en su composici√≥n de amino√°cidos, lo que podr√≠a revelar rasgos funcionales particulares o pistas sobre su historia evolutiva. üåê

calculate_amino_acid_frequencies <- function(file) {
  fasta <- read.fasta(file, seqtype = "AA")  # Leer archivo multifasta de amino√°cidos

  amino_acids <- c("A", "R", "N", "D", "C", "Q", "E", "G", "H", "I", "L", "K", "M", "F", "P", "S", "T", "W", "Y", "V")

  # Procesar cada secuencia en el archivo
  results <- lapply(names(fasta), function(seq_id) {
    seq <- toupper(getSequence(fasta[[seq_id]], as.string = TRUE)[[1]])  # Convertir a texto y may√∫sculas
    aa_counts <- table(factor(strsplit(seq, "")[[1]], levels = amino_acids))  # Contar amino√°cidos con niveles

    total_aa <- sum(aa_counts)  # Total de amino√°cidos en la secuencia

    # Crear un dataframe de frecuencias por amino√°cido
    freq_df <- data.frame(AminoAcid = names(aa_counts), 
                          Frequency = as.vector(aa_counts) / total_aa)

    # Pivotar el dataframe para tener amino√°cidos como columnas
    freq_wide <- freq_df %>%
      pivot_wider(names_from = AminoAcid, values_from = Frequency, 
                  values_fill = list(Frequency = 0))  # Llenar valores faltantes con 0

    # Agregar columnas con el nombre del archivo y el ID de la secuencia
    freq_wide <- freq_wide %>%
      mutate(File = basename(file), Sequence_ID = seq_id) %>%
      relocate(File, Sequence_ID)  # Asegurar el orden de las columnas

    return(freq_wide)
  })

  # Combinar resultados de todas las secuencias en un archivo
  bind_rows(results)
}

plot_barplot_contrast <- function(data, plot_title, y_label) {
  dodge <- position_dodge(width = 0.6)
  
  ggplot(data, aes(x = AminoAcid, y = mean_freq, fill = supp)) +
    geom_bar(stat = "identity", alpha = 0.7, width = 0.5, position = dodge) +
    geom_errorbar(aes(ymin = mean_freq - sd_freq, ymax = mean_freq + sd_freq, alpha = 0.3),
                  position = dodge, width = 0.2) +
    ylab(y_label) +
    labs(title = plot_title) +
    theme_minimal() +
    theme(
      axis.line.y = element_blank(),
      axis.ticks.y = element_blank(),
      axis.text.x = element_text(angle = 45, hjust = 1)
    )
}

4.4 ‚ú® Agrupaci√≥n de amino√°cidos por propiedades qu√≠micas

Para complementar el an√°lisis individual de los amino√°cidos, realizamos una agrupaci√≥n basada en sus propiedades qu√≠micas ‚öóÔ∏è. Esta clasificaci√≥n permite observar patrones m√°s amplios en la composici√≥n de las prote√≠nas, agrupando los amino√°cidos seg√∫n caracter√≠sticas como su carga, polaridad, hidrofobicidad o presencia de anillos arom√°ticos üî¨.

En este caso, agrupamos los amino√°cidos en las siguientes categor√≠as:

üß™ Alif√°ticos: Gly, Ala, Val, Leu, Ile

üå∏ Arom√°ticos: Phe, Trp, Tyr

üå± Con azufre: Cys, Met

‚ö° B√°sicos (carga positiva): Lys, Arg, His

üçã √Åcidos (carga negativa o derivados): Asp, Glu, Asn, Gln

üíß Polares sin carga: Ser, Thr, Asn, Gln, Tyr

üß∑ Alif√°ticos hidroxilados: Ser, Thr

üß≤ Cargados positivamente: Lys, Arg, His

üß™ Cargados negativamente: Asp, Glu

‚òÄÔ∏è Hidrof√≥bicos: Ala, Val, Ile, Leu, Met, Phe, Trp, Gly, Pro, Cys

üí¶ Hidrof√≠licos: Arg, Asn, Asp, Gln, Glu, Lys, Ser, Thr, Tyr, His

Esto nos permite comparar, por ejemplo, si la prote√≠na DGF-1 presenta una preferencia por residuos hidrof√≥bicos o cargados, en relaci√≥n con el conjunto general de prote√≠nas codificantes del genoma de Trypanosoma cruzi üß¨.

A continuaci√≥n, se muestran los gr√°ficos comparativos para cada genoma analizado:

classification_aminoacids <- function(data, genome_col = "File", id_col = "Sequence_ID", type = NULL) {
  result_combained_data <- data.frame(
    "Genome" = data[[ genome_col]],
    "id" = data[[ id_col]],
    "Aliphatic" = rowSums(data[, c("G", "A", "V", "L", "I")]),
    "Aromatic" = rowSums(data[, c("F", "W", "Y")]),
    "Sulphur" = rowSums(data[, c("C", "M")]),
    "Basic" = rowSums(data[, c("K", "R", "H")]),
    "Acidic" = rowSums(data[, c("D", "E", "N", "Q")]),  # nota: N y Q a veces no se incluyen como √°cidos
    "Polar_Uncharged" = rowSums(data[, c("S", "T", "N", "Q", "Y")]),  # Polar pero sin carga
    "AliphaticHydroxyl" = rowSums(data[, c("S", "T")]),
    "Positive_Charge" = rowSums(data[, c("K", "R", "H")]),  # Lisina, Arginina, Histidina
    "Negative_Charge" = rowSums(data[, c("D", "E")]),       # Aspartato, Glutamato
    # "tRNAsynthetaseClassI" = rowSums(data[, c("E", "Q", "R", "C", "M", "V", "I", "L", "Y", "W")]),
    # "tRNAsynthetaseClassII" = rowSums(data[, c("G", "A", "P", "S", "T", "H", "D", "N", "K", "F")]),
    "Hydrophobic" = rowSums(data[, c("A", "V", "I", "L", "M", "F", "W", "G", "P", "C")]),
    "Hydrophilic" = rowSums(data[, c("R", "N", "D", "Q", "E", "K", "S", "T", "Y", "H")])
  )
  return(result_combained_data)
}
